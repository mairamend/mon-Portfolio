À propos du projet

Ce projet de Landing a été developpé pour offrir une expérience modernes et intuitive. Le site integre toutes les fonctionnalités essentielles d'un landing page.

L'application a été construite avec une architecture moderne utilisant HTML et CSS.Le design responsives garantit une expérience optimale sur tous les appareils

Fonctionnalités

Accueil

une image avec un texte qui flotte dessus et un boutton qui permet de contacter le garage

Services

Presentes l'ensemble des services qu'offre le garage sous forme de cards

Contact

Affiche les information pour contacter le garage et un formulaire pour ecrire un message
