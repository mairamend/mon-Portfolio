À propos du projet

Ce projet d'une application de gestion d'un pressing a été developpé pour facilité la vie .Elle permet l'enregistrement d'une nouvelle commande en renseignant le nom du client son numero le nombre de vetement dans la commandes et calcule le prix puis l'enregistre dans la base de données
